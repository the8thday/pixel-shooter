# Drone Hunt 像素第三人称射击

A top-down 2D pixel art shooter built with vanilla JavaScript and HTML5 Canvas. No dependencies.

## Play

Open `index.html` in a modern browser, or serve locally:

```bash
python -m http.server 8000
```

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Aim |
| Left Click | Shoot |
| Shift | Dodge roll |
| 1-5 / Scroll | Switch weapon |

## Weapons

| # | Weapon | Style |
|---|--------|-------|
| 1 | 手枪 (Pistol) | Balanced all-rounder |
| 2 | 霰弹枪 (Shotgun) | Close-range spread |
| 3 | 冲锋枪 (SMG) | High fire rate |
| 4 | 狙击枪 (Sniper) | Long-range, high damage |
| 5 | 火箭筒 (Rocket) | Explosive AoE |

## Game Content

- **10 levels** with unique themed maps (Volcano, Grassland, Mystic Forest, Ocean, Desert, Abandoned City, Space, Battlefield, Bone Forest, Factory)
- **Boss fights** every 3rd level
- **Super bosses** on levels 5 and 10 with phase-based mechanics
- Procedurally generated maps, music, and pixel art visuals
- Minimap, auto-hide HUD, screen shake, and particle effects
